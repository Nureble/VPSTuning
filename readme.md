# Network Stack Optimizer for High-Load Systems

Скрипт для автоматической тонкой настройки сетевых параметров Linux (Ubuntu/Debian). Предназначен для оптимизации серверов с высокой пропускной способностью: веб-серверы, стриминг-платформы, базы данных и системы доставки контента.

---

## Возможности и механика

- **TCP BBR** — активирует современный алгоритм контроля перегрузок от Google, который значительно увеличивает скорость передачи данных на нестабильных каналах.
- **Диспетчер очередей FQ** — переводит сетевой интерфейс на работу с Fair Queueing, что снижает джиттер и является обязательным для эффективной работы BBR.
- **Автоматический расчёт MTU** — вычисляет максимально допустимый размер пакета без фрагментации для вашего провайдера, что снижает нагрузку на CPU и уменьшает потери пакетов.
- **Тюнинг сетевых буферов** — расширяет лимиты памяти `rmem` и `wmem` до 32 МБ для поддержки «толстых» каналов связи (LFN).
- **Оптимизация TCP-стека** — отключает повторный slow-start после простоя соединения и сокращает `fin_timeout` для быстрого освобождения сокетов под высокой нагрузкой.
- **Security Stealth** — отключает ответы на входящие ICMP (ping) запросы, делая сервер менее заметным для автоматизированных сканеров сети.

## Использование

### Автоматически (одной командой)

```bash
curl -sSL https://raw.githubusercontent.com/Nureble/VPSTuning/refs/heads/main/vpstuning.sh | sudo bash
```

### Вручную

```bash
git clone https://github.com/Nureble/VPSTuning.git
cd VPSTuning
chmod +x vpstuning.sh
sudo ./vpstuning.sh
```

---

## Требования

- **Операционная система:** Debian 11+ или Ubuntu 20.04+
- **Права:** root или sudo
- **Виртуализация:** KVM, VMware или физический сервер (OpenVZ/LXC поддерживаются ограниченно)

## Проверка результата

После выполнения скрипта можно убедиться, что настройки применились:

```bash
# Проверка алгоритма контроля перегрузок (должно вернуть "bbr")
sysctl net.ipv4.tcp_congestion_control

# Проверка qdisc на основном интерфейсе (должен быть "fq")
tc qdisc show

# Проверка применённого MTU
ip link show
```
